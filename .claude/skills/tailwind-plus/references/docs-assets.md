# Assets Documentation

> Source: https://tailwindcss.com/plus/ui-blocks/documentation/assets

## Overview

Documentation of third-party assets and resources used throughout Tailwind Plus examples. This is a reference page for asset attribution and sources.

---

## Assets

### 1. Icons - Heroicons

- **Source:** [heroicons.com](https://heroicons.com)
- **License:** Free MIT-licensed
- **Description:** All icons used in Tailwind Plus come from Heroicons, designed and developed by the Tailwind team
- **Installation:**
  ```bash
  npm install @heroicons/react
  ```
- **Usage:**
  ```jsx
  import { ChevronDownIcon } from '@heroicons/react/20/solid'
  import { Bars3Icon } from '@heroicons/react/24/outline'
  ```

### 2. Images - Unsplash

- **Source:** [unsplash.com](https://unsplash.com)
- **License:** Free/Freely-usable
- **Description:** Images in Tailwind Plus come almost exclusively from Unsplash
- **Usage:** Free photography resources for placeholder images in examples

### 3. Illustrations - Lucid Illustrations

- **Source:** [lucid.pixsellz.io](https://lucid.pixsellz.io) by [Pixsellz](https://www.pixsellz.io)
- **License:** Free
- **Description:** Some examples use illustrations from the free Lucid Illustrations pack

### 4. Figma Assets (DISCONTINUED)

- **Status:** Discontinued as of July 14, 2021
- **Notes:**
  - Extremely high maintenance burden
  - Low usage rate
  - No longer receives updates
  - Does not contain examples released after July 14, 2021
- **Download:** Still available for Tailwind Plus customers at [/plus/download-figma](/plus/download-figma)
